# Моя фильмотека

Личный каталог просмотренных фильмов, сериалов и аниме. Статичный сайт для GitHub Pages — чистый HTML/CSS/JS, без сборки.

## Структура

- `index.html` — главная: вкладки «Главная / Топ 100 / Новинки», фильтры, карточки-строки
- `film.html?id=...` — страница одного произведения
- `admin.html` — локальный инструмент добавления (не публикуется в меню)
- `data.json` — база записей
- `css/style.css`, `js/app.js`, `js/film.js`, `js/admin.js`

## Как добавить фильм

1. Открыть `admin.html` (локально или на сайте — ссылки на него в меню нет).
2. Один раз вписать бесплатные ключи [TMDb](https://www.themoviedb.org/settings/api) и [OMDb](https://www.omdbapi.com/apikey.aspx) — они хранятся только в localStorage браузера.
3. Найти произведение по названию → проверить предпросмотр → поставить свою оценку, комментарий, статус и тип.
4. «Скопировать JSON-запись» → вставить в массив в `data.json` (или «Скачать обновлённый data.json» и заменить файл).
5. Закоммитить и запушить — сайт обновится автоматически.

## Публикация

GitHub → Settings → Pages → Source: Deploy from a branch → ветка `main`, папка `/ (root)`.
